Feature Embedding by Template Matching as a ResNet Block
(arXiv)
A. Gorgun, Y. Z. Gurbuz, A. A. Alatan, British Machine Vision Conference (BMVC), 2022

We revamp 3x3-BN-ReLU-1x1 based feature transform to assign a feature vector to each local region according to the semantic meaning of the template that the corresponding region matches. Specifically, we show that batch normalization (BN) followed by a rectified linear unit (ReLU) can be interpreted as arg-max optimizer. Following this perspective, we tailor a residual block that explicitly forces semantically meaningful local feature embedding through using label information. We evaluate our method on three popular benchmark datasets with several architectures for image classification and consistently show that our approach substantially improves the performance of the baseline architectures.
